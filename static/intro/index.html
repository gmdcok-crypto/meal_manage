Features

핵심 기능

  • 📱

    직원 PWA

    최초 1회 기기 인증 후, 현장 QR 스캔으로 식사 인증. 인증 완료 화면으로 배식 담당자에게 바로 확인 가능.

  • 🖥️

    PC 관리자

    회사·부서·사원 관리, 식사 정책, 당일·기간별 보고서, 공지사항, 식당관리자(위탁사) 계정 관리. 실시간 WebSocket 대시보드.

  • 📋

    당일 식사인증 조회

    식당관리자 전용 PWA. 사번·이름 검색으로 오늘(한국시간) 식사 인증 여부·시간·식사 종류를 즉시 조회.

  • 🖨️

    식권 출력·경광등

    네트워크 프린터(빅솔론 등)로 식권 자동 출력, Q라이트 경광등 연동으로 인증 시 시각·음향 알림(선택 구성).
